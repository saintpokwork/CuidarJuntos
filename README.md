# CuidarJuntos

Plataforma de organização familiar para cuidadores em Portugal. Exportada do Stitch e adaptada para React.

## Estrutura

- `src/pages/Home.tsx` — Página de destino (landing)
- `src/pages/Dashboard.tsx` — Painel principal
- `src/components/` — Componentes partilhados
- `src/styles/` — Estilos globais (Tailwind CSS)
- `stitch_landing_page_cuida.pt/` — Referência de design original (Stitch)

## Executar localmente

```bash
npm install
npm run dev
```

A aplicação abre em [http://localhost:3000](http://localhost:3000).

## Build de produção

```bash
npm run build
```

## Rotas

| Rota | Página |
|------|--------|
| `/` | Landing page |
| `/dashboard` | Painel principal |
